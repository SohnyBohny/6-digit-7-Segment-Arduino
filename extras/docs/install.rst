How to install & include the library
==========================
1. Download via LibraryManger
--------------------------
* Open Arduino IDE
* goto ``Sketch`` - ``Include Library`` - ``Manage Libraries...``
* install six-digit-seven-segment

2. Include Library
------------------
* add library ``Sketch`` - ``Include Library`` - ``six-digit-seven-segment``
* `check how to use`_
